# SETコマンド仕様書 (案)

## 概要

ユーザー入力からキーと値を抽出し、ストアに保存する。

## 形式

`SET <key> <value>`

- 例: `SET user_name rust_ace`

## パース処理の挙動

1. 入力文字列を `split_whitespace().collect::<Vec<_>>()` で分割し、`Vec` に格納する。
2. `Vec` の 0 番目の要素が厳密に `"SET"` であることを確認する。
3. `Vec` の 1 番目の要素を `key`、2 番目の要素を `value` とする。

## エラーケースと対応

- `Vec` の要素数が 3 未満の場合
  - 対応: 「エラー：引数が足りません」という旨のメッセージを表示する。
- 0 番目の要素が `"SET"` 以外の場合
  - 対応: 無視するか、別のコマンド（GET等）の処理へ回す。

## 期待される内部データ構造（案1：初級）

- `HashMap<String, String>` に保存される。
